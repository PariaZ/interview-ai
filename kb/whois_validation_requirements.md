# WHOIS Validation Requirements

## 1. Required WHOIS Data
Registrants must provide accurate:
- Full legal name
- Physical mailing address
- Valid email address
- Phone number with country code

## 2. Verification Triggers
Verification may be required when:
- WHOIS data appears incorrect
- Abuse reports reference inaccurate WHOIS
- ICANN-mandated audits occur
- A transfer or ownership change is initiated

## 3. Section 4.2 — Verification Deadline
Registrants must respond within **72 hours** of receiving a verification 
notice.

Failure to respond results in:
- Temporary suspension
- DNS disruption
- Escalation to Compliance

## 4. Completing Verification
To complete WHOIS verification:
1. Update inaccurate WHOIS fields  
2. Respond to verification email  
3. Provide identification documents if requested  
4. Confirm phone number if required  

## 5. Clearing a Suspension
Once WHOIS information is validated:
- Support confirms compliance
- The domain is reinstated
- DNS is restored within 15–30 minutes
